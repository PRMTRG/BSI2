# Python Examples of encryption/decryption

## Simple python project created to test out possiblities of:
* ### AES
* ### Blowfish
* ### Triple DES
----
                    
## Input Files
                    

| File Name | Description                    |
| ------------- | ------------------------------ |
| **3des_key.txt**      | Storage file for TripleDES key       |
| **aes_key.txt**   | Storage file for AES key      |
| **blowfish_key.txt**   | Storage file for BlowFish key      |
| **message.txt / message2.txt**   | Storage files for example messages to encrypt      |
